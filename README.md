### 其他功能

* 插入排序与折半排序后可以选择进行查找算法
* 统计排序运行时间

### In/Out

* 控制台输入输出
* 文件流输入输出
  * 待排序数据存放在`D://`下的`sortlist.txt`
  * 格式要求：第一行给出数据数量，第二行空格间隔输入每个数
